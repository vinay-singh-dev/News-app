# NewsApp ðŸ“°

You can find the news by searching. You can look at the details of the news, you can go to the news site. You can add or remove news from your favorites. You can share the news.

<br>

When the Api Key exceeds the limit or there is a problem with the internet, a "Network error" message appears on the screen.

When there is no result for the related search, the message "There are no results" is displayed.

<br>
